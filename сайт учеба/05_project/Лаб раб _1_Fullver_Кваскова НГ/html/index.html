Пояснительная записка

Задача: создать веб-сайт, включающий перечисленные операции:

  1. Создайте пустой HTML-файл.
  2. Создайте в этом файле базовую структуру HTML-документа.
  3. Задайте произвольный заголовок страницы (title).
  4. Напишите заголовок первого уровня в любым текстом.
  5. Напишите заголовок третьего уровня.
  6. Создайте параграф длиной не менее 1000 символов.
  7. Выделите первое слово ранее созданного параграфа жирным шрифтом, а последнее курсивом.
  8. Создайте ненумерованный список городов-миллионников России.
  9. Создайте нумерованный список ингридиентов вашего любимого блюда.
  10. Создайте ссылки на все ваши профили в социальных сетях. Одна из ссылок должна открываться в новом окне браузера.
  11. Возьмите несколько картинок, положите в одну папку с вашим скриптом и отобразите на странице.
  12. Задайте высоту картинок (например 200 пикселей).
  13. Напишите таблицу, состоящую из четырех столбцов и трех строк. Заполните произвольными данными.
  14. Создайте форму. Укажите, что форма будет обрабатываться этим же скриптом.
  15. В форме создайте элементы для ввода имени, фамилии, электронного адреса, пароля, а также описание к этим полям.
  16. После поля с паролем в форме добавьте элемент выбора пола используя радио кнопки (м, ж).
  17. Далее в форме добавьте выпадающий список, состоящий из произвольных элементов.
  18. Добавьте ранее созданному списку возможность множественного выбора значений.
  19. Ниже в форме добавьте текстовое поле размером 5 строк и 40 колонок.
  20. После текстового поля добавьте элемент для прикрепления файла.
  21. Последним элементом формы добавьте кнопку ее отправки.
  22. * В качестве опционального задания попробуйте стилизовать страницу используя самостоятельный поиск информации по CSS.
